# 项目说明

1. `web-admin`是后台管理系统项目
2. `web-company`是公司门户网站项目
3. `web-server`是服务端

## 安装

```
npm install
```

## 运行

1. 前端项目(后台管理系统和门户官网)

```
npm run serve
```

2. 服务端启动

```
npm start
```
